# Mults Parser - Android AppAndroid приложение для просмотра каталога мультфильмов с сайта mults.info.## ?? Возможности- Просмотр каталога мультфильмов- Быстрый доступ к страницам мультфильмов- Простой и удобный интерфейс## ??? Сборка1. Клонируйте репозиторий2. Откройте проект в Android Studio3. Нажмите Build > Generate Signed Bundle / APK## ?? Скачать APKГотовый APK файл можно скачать из раздела [Releases](https://github.com/your-username/mults-parser-android/releases) или из [Actions](https://github.com/your-username/mults-parser-android/actions).## ?? ЛицензияЭтот проект предназначен только для образовательных целей.
